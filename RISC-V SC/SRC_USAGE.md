- **SC_04**  
  Contains files for:
  - `top_coprocessor_control`
  - `ISO14443_wrapper`

- **RV-SC02**  
  Contains files for a smartcard using PicoRV32 with:
  - 256 KB ROM
  - The necessary components

- **RV-SC03**  
  Contains files for a smartcard using PicoRV32 with:
  - 512 KB ROM
  - The necessary components

## Usage
Use **RV-SC02** or **RV-SC03** together with **SC_04**.
